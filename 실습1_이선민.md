## Pre-qualification
### Release SELinux
https://www.lesstif.com/pages/viewpage.action?pageId=6979732
~~~
sudo vi /etc/sysconfig/selinux >> SELINUX=disabled
reboot
~~~

### Hosts Setting
~~~
# Setting host name
# m1은 변경하고자 하는 이름
[hadoop@localhost ~]$ hostnamectl set-hostname m1
[hadoop@localhost ~]$ hostname
m1

# n1, n2 에서도 동일한 작업 진행
~~~
